    We can wrap \texttt{\textbackslash DrawCommand} with \texttt{adjustbox} (from package \texttt{adjustbox}) to push inline hand's baseline a bit lower:

    Like this: \adjustbox{margin*=0ex -0.5ex 0ex 0ex}{\DrawHand{p123345567s4478~9}{1.5em}}

    Or like this: \adjustbox{margin*=0ex -1ex 0ex 0ex}{\DrawHand{p123345567s4478~9}{1.5em}}
